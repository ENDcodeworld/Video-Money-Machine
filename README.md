## 🚀 快速开始

### 安装

```bash
git clone https://github.com/ENDcodeworld/Video-Money-Machine.git
cd Video-Money-Machine

```

### 运行

```bash

python run.py --count 5 --make-video --plan
```

## 🎬 视频制作流程

### 1. 准备素材

```bash
# 创建素材目录
mkdir -p assets/images assets/music

# 放入素材
# - 图片：1080x1920 竖屏
# - 音乐：MP3格式，30-120秒
```

**免费素材网站：**
- 图片：Unsplash, Pexels, Pixabay
- 音乐：YouTube Audio Library, Free Music Archive

### 2. 生成视频

```bash
# 生成制作命令
python run.py --make-video

# 查看命令
cat output/video_commands/video_001_command.txt

# 执行FFmpeg命令（需安装FFmpeg）
ffmpeg [命令参数]
```

### 3. 替代方案

如果没有素材，可以用：
- **AI绘图**：Midjourney / Stable Diffusion
- **录屏**：PPT讲解 / 屏幕录制
- **混剪**：电影片段混剪（注意版权）
- **动画**：文字动画 / 数据可视化
